# tarifas2026

Web sencilla (HTML/CSS/JS) para calcular totales de factura con IVA y (si aplica) retención IRPF.

## Uso
- Abre `index.html` (Live Server / doble clic).
- Elige tipo de cliente:
  - Empresa/autónomo: calcula IRPF y lo resta del total.
  - Particular: oculta IRPF y calcula solo base+extras+IVA.

> Nota: herramienta orientativa; revisa tu caso con asesoría si tienes exenciones/IVA especial.
